## Description

[Link on Client](https://github.com/Marshalleye/telegram-bot-client)
<h1> It's a Telegram bot that can track the reputation of every bot member </h1>

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Init DB
```bash
npx prisma migrate dev --name init 
```
